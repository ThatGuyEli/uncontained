addLeaderboardEntry(levelid, initials, score)

Add a leaderboard entry to the leaderboard file. Note that this file is not stored within the app, but in the user's local appdata directory. On Windows: C:\Users\{youruser}\AppData\Roaming\ On Mac: ~/Library/Application Support/ On Linux: ~/.config/
Parameters:
Name Type Description
levelid number The ID of the level to add an entry for.
initials string A three-character string to attach to the entry.
score number The score of the entry.
